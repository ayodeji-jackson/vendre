# Vendre.
e-commerce web app. Bringing the design i found on [dribbble](https://dribbble.com/) to life with stuff. Live on vercel @ https://vendre-weld.vercel.app/

## features
- [ ] PWA
- [ ] Dark mode
- [x] Filter by category, price range, search phrase
- [x] Sort by price
- [x] Add to wishlist
- [x] Add to cart
- [ ] Payment
